# README

## 사용법이니 꼭 읽어주세요

1. requirements.txt 에 작성된 패키지들을 다운로드 받는다.
2. image 폴더에 원하는 사진을 담는다. 
3. image_split.py에서 8번째 줄 image_name을 자신의 사진 이름으로 변경한다. 이름에 공백을 제거하는 것을 추천합니다.   
4. .env에 본인의 메타모스트 아이디, 비밀번호, 주소 입력 - 메타모스트 주소는 메타모스트 앱 좌측 최상단 서버 명 클릭 서버 수정 버튼 클릭시 나타남
5. 이제 image_split 파일을 실행하면 됩니다. 
6. 이때 이름을 작성하라 하는데 이것이 이모지의 이름이 됩니다. 
7. output 폴더에 이미지가 쪼개져 생성됩니다. 완료가 되었다면 image_upload.py 를 실행하면 업로드 됩니다.  
8. AI한테 ```텍스트 형태로 한번에 복사되게 가로는 다 붙여서 행 변경시는 엔터로 :name_a1: 부터 :name_f13: 까지 알파벳 abcd를 행으로 숫자 1234를 열로 해서 만들어줘``` 라고 부탁후 해당내용 메타모스트에 입력하면 됩니다. - 각 가로 이모지 명령어 사이에는 공백이 있으면 안됩니다. :name_a1:::name_a2: 처럼 붙어있어야 함. 범위는 생성된 output을 보고 바꾸면 됩니다.

주의사항  
1. 이름을 emoji_name으로 입력시 emoji_name_a1 emoji_name_a2 ... emoji_name_f10 과 같은 형태로 이뤄집니다.
2. 동일한 이름이 하나라도 있으면 모두 업로드 되지 않고 실패하니 동일한 이름이 없는지 확인하기 바랍니다.  
3. 재사용시 output 폴더는 삭제하고 다시 하셔야 동일한 이미지를 재업로드 하지 않습니다. 
4. 삭제하지 않을 시 똑같은 이름으로 또 올릴거라 에러뜨고 멈춥니다.
5. **이미지 업로드는 매크로로 하는거라 정책위반일수도..?  ** - 확인 안 해봄 책임 안 짐
